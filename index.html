Hello Daarling is a font exploring the rough edges of a worn out love. How people get tired of each other, but still try to express themselves with care. Inspired by the CSS Sans by Yusukue Sugomori, it is created using CSS and HTML. The font is constructed by combining pre-made colour blocks.

Hello Daarling is a mainly meant for writing short phrases, but longer texts are not excluded. To write, the source code has to be downloaded and added to the original code. Each letter is created separately so it is just simply to copy and paste them onto your project.
